# ascii-buddy

A terminal Tamagotchi companion that lives in your CLI as animated ASCII art, speaks developer wisdom generated by Gemini AI, and persists between sessions.

```
╔═══════════════╗
║               ║  "Premature optimization is the root of all evil.
║   /\  /\      ║   So is premature abstraction."
║  ( o)(o)      ║
║   \/\/        ║   — Crash (null pointer survivor)
║   /||\\       ║
║  (_||_)       ║   [offline]  last seen: just now
╚═══════════════╝
 [n] new phrase   [q] quit
```

## Requirements

- Node.js ≥ 18
- A [Gemini API key](https://aistudio.google.com/app/apikey) (free tier works; offline fallback phrases available without one)

## Install

```bash
npm install -g ascii-buddy
```

Or run directly without installing:

```bash
npx ascii-buddy
```

## Usage

```bash
# Start the companion (uses your active buddy)
ascii-buddy

# Create a new AI-generated buddy
ascii-buddy --new

# Browse and switch between buddies
ascii-buddy --list

# Open the buddy picker to delete a buddy (press [d] in the list)
ascii-buddy --delete
```

## Configuration

Set your Gemini API key via environment variable:

```bash
export GEMINI_API_KEY=your_key_here
ascii-buddy
```

Or create `~/.ascii-buddy/config.json`:

```json
{
  "apiKey": "your_key_here",
  "provider": "gemini",
  "phraseIntervalSeconds": 30
}
```

### Config options

| Key | Default | Description |
|-----|---------|-------------|
| `apiKey` | — | Gemini API key (falls back to `GEMINI_API_KEY` env var) |
| `provider` | `"gemini"` | AI provider (only `"gemini"` supported in v1) |
| `phraseIntervalSeconds` | `30` | How often your buddy speaks automatically |

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `n` | Generate a new phrase immediately |
| `q` / `Ctrl+C` | Quit |

## Offline mode

If no API key is set, ascii-buddy runs in offline mode with 50+ built-in dev phrases categorised by tone (motivational, sarcastic, wise, funny, roast). Your buddy shows a `[offline]` indicator and goes **sad** when AI is unavailable.

## Mood states

| Mood | Trigger |
|------|---------|
| IDLE | Default — buddy is waiting |
| TALKING | Phrase is being generated |
| HAPPY | Phrase generated successfully (5s) |
| SAD | AI unavailable / phrase failed (5s) |
| SLEEPING | No keypress for 10 minutes |
